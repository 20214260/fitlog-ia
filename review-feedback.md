## 5️⃣ 정보 우선순위 / 위계 분석

### 🟠 P07 홈 — 정보 위계 문제

**현황**:
- 오늘의 요약 카드 (70px 높이)
- AI 추천 루틴 카드 (80px 높이)
- 최근 운동 리스트 (3개, 약 130px 차지)

**문제**:
- 명세서에 따르면 **AI 추천이 서비스 차별화 포인트**이지만, 시각적 비중이 오늘의 요약과 비슷
- 최근 운동 리스트가 첫 화면에서 너무 많은 공간 차지 → 스크롤 부담
- "시작하기" CTA가 작아서 핵심 동선이 약함

**개선 제안**:
1. AI 추천 카드 높이 **80px → 120px**로 확대, 일러스트/그라데이션 추가
2. "시작하기" 버튼 **40px → 48px**로 확대, 색상 강조
3. 최근 운동 리스트는 **3개 → 2개**로 축소, "전체 보기" 링크로 P18 캘린더 유도

### 🟠 P12 운동 완료 요약 — 핵심 CTA 위계 문제

**현황**:
- 하단 3개 버튼: 공유 / 통계 / 홈 모두 동일한 크기와 색상
- 홈 버튼만 파란색이지만 너비는 동일

**문제**:
- 시나리오 분석상 **"홈으로"가 가장 빈도 높은 동선**이지만 시각적으로 강조되지 않음
- 공유 버튼은 사용 빈도가 낮은데 동일 위계로 배치

**개선 제안**:
1. "홈으로" 버튼 **전체 너비의 60% 차지**하도록 확대, 메인 컬러 적용
2. 공유는 **상단 아이콘 버튼**으로 이동 (헤더 우측)
3. 통계 보기는 결과 카드 하단에 텍스트 링크로 배치

### 🟡 P11 운동 상세 — 시선 흐름 개선

**현황**: 운동명 / 자세 이미지 / 세트 리스트 / 운동 완료 버튼 순서

**개선 제안**:
- 자세 이미지를 **접을 수 있는(collapsible) 카드**로 변경 → 세트 입력에 집중 가능
- 휴식 타이머는 **세트 완료 시 화면 중앙에 풀스크린 오버레이**로 표시 (현재는 작은 텍스트)
